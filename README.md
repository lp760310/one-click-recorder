# 个人学习中心 · 一键录音记录

一个极简的纯前端录音记录网页工具，用于随手记录学习过程中的语音想法、复盘片段和临时备注。

## 本地打开方式

直接用浏览器打开 `index.html` 即可预览页面。

手机真实录音建议通过 HTTPS 地址访问，例如 GitHub Pages：

```text
https://lp760310.github.io/one-click-recorder/
```

原因是多数手机浏览器要求 HTTPS 页面才能请求麦克风权限。

## 功能说明

- 一键开始录音 / 停止录音
- 录音时显示“正在录音”
- 停止后自动生成一条记录
- 每条记录包含日期时间、音频播放控件、文字备注输入框和删除按钮
- 录音和备注保存在当前浏览器的 IndexedDB 中
- 支持导出全部记录为 JSON
- 支持导入 JSON 恢复记录

## 隐私说明

当前版本没有后端服务，不连接数据库，也不会上传录音文件。录音只保存在当前浏览器本地。导出的 JSON 文件由用户自行保存。
