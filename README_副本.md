# 一键录音

一个极简的纯前端录音网页工具，用于连续累计同一个当前语音文件，并在需要时导出音频文件。

## 本地打开方式

直接用浏览器打开 `index.html` 即可预览页面。

手机真实录音建议通过 HTTPS 地址访问，例如 GitHub Pages：

```text
https://lp760310.github.io/one-click-recorder/
```

原因是多数手机浏览器要求 HTTPS 页面才能请求麦克风权限。

## 功能说明

- 一键开始录音 / 停止录音
- 停止后保留当前语音缓存，再次录音会追加到上一段后面
- 多次录音、停止、继续后，只导出一个完整音频文件
- 导出成功后清空当前语音缓存，再次录音会开始新的语音文件

## 隐私说明

当前版本没有后端服务，不连接数据库，也不会上传录音文件。语音只保存在当前页面内存中，导出前不会和学习记录混在一起。
